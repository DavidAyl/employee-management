## About

This JavaScript application takes use of MySql to store information about a team in a database. This application has simplicity as all the user has to do is use inquirer. With these prompts you can create, update, and view, multiple things like Departments, Roles, and employeees.

<hr>


## How To Use

This program thrives on it's user friendly interface and simplicity. First you will run a `npm install` command in the terminal/console. After this you can run the final command `node index.js`.
